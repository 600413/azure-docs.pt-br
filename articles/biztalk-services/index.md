# <a name="biztalk-services-documentation"></a>Documentação dos Serviços BizTalk

Os Serviços BizTalk fornecem integração de aplicativos da nuvem com o sistema local e a de linha de negócios para SAP, Oracle EBS, SQL Server e PeopleSoft. Eles permitem que você se conecte a qualquer fonte de dados HTTP, FTP, SFTP ou REST. Você pode rotear mensagens usando vários artefatos do Azure, como Filas do Barramento de Serviço, Tópicos, Banco de dados SQL e Armazenamento de blobs.
