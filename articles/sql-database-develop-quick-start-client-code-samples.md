# Desenvolvimento do cliente e exemplos de código de início rápido para o Banco de Dados SQL


Este tópico fornece links para exemplos de código de início rápido que você pode usar para se conectar ao Banco de Dados SQL do Azure. Os exemplos abrangem:


- Diversas linguagens de programação.
- O Windows e o Linux como o sistema operacional no qual programa cliente será executado.
- Exemplos curtos de código de início rápido, além de exemplos mais longos que tratam de erros temporários com uma lógica de repetição automática.
- Exemplos de código que convertem os conjuntos de resultados relacionais em um formato orientado ao objeto.


> [AZURE.NOTE]Desde 19 de maio de 2015, exemplos de código para outras linguagens estão sendo preparados, e os links até eles serão adicionados a este tópico.
