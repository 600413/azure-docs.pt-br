# Tamanhos das máquinas virtuais do Windows no Azure

Este artigo descreve os tamanhos e as opções disponíveis de máquinas virtuais do Windows que você pode usar para executar seus aplicativos e cargas de trabalho. Ele também fornece considerações de implantação a serem observadas ao planejar o uso desses recursos. Para obter informações sobre os preços dos vários tamanhos, consulte [Preços de máquinas virtuais](https://azure.microsoft.com/pricing/details/virtual-machines/). Se você quiser obter informações sobre as VMs do Linux, confira a seção [Sizes for Linux virtual machines in Azure](virtual-machines-linux-sizes.md) (Tamanhos das máquinas virtuais do Linux no Azure)
