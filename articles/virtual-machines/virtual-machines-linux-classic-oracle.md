# Software Oracle hospedado em máquinas virtuais do Linux no Azure

[AZURE.INCLUDE [virtual-machines-common-oracle-support](../../includes/virtual-machines-common-oracle-support.md)]

Configure seu primeiro ou próximo banco de dados de desenvolvimento/teste ou de produção Oracle para ser executado em uma máquina virtual Linux nos serviços de infraestrutura do Azure. Tire proveito da facilidade de configuração e da capacidade de expandir rapidamente a carga de trabalho do banco de dados Oracle para incluir a nova capacidade.
