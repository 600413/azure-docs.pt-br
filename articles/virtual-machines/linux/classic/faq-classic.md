> [!IMPORTANT] 
> O Azure tem dois modelos de implantação diferentes para criar e trabalhar com recursos: [Gerenciador de Recursos e Clássico](../../../resource-manager-deployment-model.md). Para as Perguntas Frequentes ao usar o modelo do Resource Manager, veja [aqui](../faq.md?toc=%2fazure%2fvirtual-machines%2flinux%2ftoc.json).
