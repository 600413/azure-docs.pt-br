# <a name="getting-started"></a>Introdução aos Serviços Móveis

Este tópico não tem mais suporte; um projeto autônomo da Windows Store não está mais disponível para download no portal. Em vez disso, você pode baixar um projeto do Windows universal. Para obter mais informações, consulte [Introdução aos Serviços Móveis](/documentation/articles/mobile-services-javascript-backend-windows-store-dotnet-get-started/). 

Para obter instruções sobre como adicionar Serviços Móveis a um aplicativo da Windows Store existente, consulte [Adicionar Serviços Móveis a um aplicativo existente](/documentation/articles/mobile-services-windows-store-dotnet-get-started-data/).
