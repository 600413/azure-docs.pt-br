#Que são os sites?

[AZURE.INCLUDE [disclaimer](../includes/disclaimer.md)]

Os sites são hosts de aplicativos Web que dão suporte a tecnologias de aplicativos Web populares, como .NET, Node.js e PHP sem exigir nenhuma alteração de código nos aplicativos existentes.  Todos os recursos e modelos de programação comuns usados pelos desenvolvedores do .NET, do Node.js e do PHP para acessar recursos, como arquivos e bancos de dados, continuarão funcionando conforme esperado nos sites implantados no Azure.  Os sites fornecem acesso a recursos de arquivos e bancos de dados e a todos os arquivos de configuração padrão (como web.config, php.ini e config.js) usados atualmente pelos aplicativos web. Um site é o host de um aplicativo web em execução na nuvem, que você pode controlar e gerenciar.
