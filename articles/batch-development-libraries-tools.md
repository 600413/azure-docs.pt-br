# Ferramentas e bibliotecas de desenvolvimento do Batch 
<p> Obtenha essas bibliotecas e ferramentas para desenvolver aplicativos do Batch do Azure do Batch apps.

+ [Batch]
+ [Batch Apps]
+ [Recursos adicionais]

## Batch
 
+ [Biblioteca de cliente Batch para .NET](http://www.nuget.org/packages/Azure.Batch/) (NuGet) - desenvolver aplicativos de cliente para executar trabalhos com o serviço Batch
+ [Biblioteca de gerenciamento do Batch](http://www.nuget.org/packages/Microsoft.Azure.Management.Batch/) (NuGet) - Gerenciar contas de serviço Batch
+ [Gerenciador do Batch](https://code.msdn.microsoft.com/windowsazure/Azure-Batch-Explorer-c1d37768) (MSDN) - criar um aplicativo de GUI para exibir e gerenciar trabalhosdo Batch

## Batch Apps

+ [SDK de nuvem no Batch Apps](http://www.nuget.org/packages/Microsoft.Azure.Batch.Apps.Cloud/) (NuGet) - permite que os aplicativos descarregam trabalhos no serviço Batch
+ [Extensão do Visual Studio no Batch Apps](https://visualstudiogallery.msdn.microsoft.com/8b294850-a0a5-43b0-acde-57a07f17826a) (Galeria do Visual Studio) - aplicativos habilitados para nuvens no Visual Studio usando o SDK de nuvem no Batch Apps
+ [SDK de cliente no Batch Apps](http://www.nuget.org/packages/Microsoft.Azure.Batch.Apps/) (NuGet) - interagir com aplicativos habilitados para descarregar trabalhos no Batch Apps
