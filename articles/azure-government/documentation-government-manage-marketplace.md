# <a name="azure-marketplace-for-government"></a>Azure Marketplace para o Azure Governamental
O Azure Marketplace está disponível para o Azure Governamental com uma lista atualizada de imagens de nossos editores do marketplace. 

## <a name="variations"></a>Variações
A seguir, algumas considerações ao usar o Azure Marketplace para o Azure Governamental:

* Somente as imagens BYOL (Traga sua Própria Licença) estão disponíveis. Você não pode acessar qualquer imagem que exija a transação de compra por meio do Azure Marketplace
* Apenas um subconjunto das imagens está disponível em comparação ao marketplace público. Você pode encontrar a lista de imagens disponíveis [aqui](../azure-government-image-gallery.md) 
* Antes de provisionar uma imagem, o Administrador Corporativo deve habilitar as compras do Marketplace para sua assinatura do Azure
  * Faça logon no Portal do como um Administrador Corporativo
  * Navegue até *Gerenciar*
  * Em *Detalhes do Registro*, clique no ícone de lápis ao lado do item de linha *Azure Marketplace*
  * Alterne *Habilitado/Desabilitado* como apropriado
  * Clique em *Salvar*

> [!NOTE]
> Se você estiver interessado em disponibilizar suas imagens no Azure Governamental, consulte as [diretrizes de integração de parceiro](documentation-government-manage-marketplace-partners.md) para saber mais.
