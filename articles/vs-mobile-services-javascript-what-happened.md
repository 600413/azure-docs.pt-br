### <span id="whathappened">O que aconteceu com meu projeto?</id>

##### Referências adicionadas

A biblioteca do Windows Azure Mobile Service foi adicionada ao seu projeto na forma de um arquivo **MobileServices.js**.

##### Valores de cadeia de conexão para Serviços Móveis

Na pasta `services\mobileServices\settings`, um novo arquivo JavaScript (.js) com um **MobileServiceClient** foi gerado contendo o URL do aplicativo do serviço móvel selecionado e uma chave do aplicativo.
