# Lista de Aplicativos de API e Conectores no Serviço de Aplicativo do Microsoft Azure

A tabela a seguir lista todos os conectores e Aplicativos de API criados pela Microsoft que estão disponíveis, em ordem alfabética: 

	Nome  | Descrição
------------- | ------------- 
Conector AS2 | O Conector de AS2 pode receber e enviar mensagens usando o protocolo de transporte AS2 em comunicações B2B.  Os dados são transportados de maneira segura e confiável pela Internet usando criptografia e certificados digitais.<br/><br/>[Conectores B2B](app-service-logic-b2b-connectors.md)
Azure HDInsight | O Azure HDInsight implanta e provisiona clusters do Apache Hadoop na nuvem e fornece uma estrutura de software criada para gerenciar, analisar e relatar Big Data.  Esse conector cria um cluster do Hadoop no Azure e envia diferentes trabalhos do Hadoop, como Pig, Hive, MapReduce e Streaming MapReduce.  Usando esse conector, você pode criar um cluster, enviar um trabalho e aguardar sua conclusão.<br/><br/>[Conectores de Aplicativo + Serviços de Dados](app-service-logic-data-connectors.md)
Serviços Móveis do Azure | Os Serviços Móveis do Azure reúnem um conjunto de serviços do Azure que habilitam recursos de back-end para aplicativos móveis.  Usando esse conector, você pode se conectar aos Serviços Móveis do Azure e criar, atualizar, obter, excluir entradas e fazer chamadas de API personalizadas em uma tabela.  <br/><br/>[Conectores de Aplicativo + Serviços de Dados](app-service-logic-data-connectors.md)
Barramento de Serviço do Azure | O Barramento de Serviço do Azure é um sistema de mensagens genérico, baseado em nuvem, para conectar praticamente qualquer coisa.  Ele habilita a troca de mensagens de maneira menos rígida, para melhor dimensionamento e confiabilidade.  Usando esse conector, você pode enviar mensagens de entidades do Barramento de Serviço, como Filas e Tópicos, e receber mensagens de entidades do Barramento de Serviço, como Filas e Assinaturas.<br/><br/>[Conectores de Aplicativo + Serviços de Dados](app-service-logic-data-connectors.md)
Blob de Armazenamento do Azure | O Blob de Armazenamento do Azure armazena grandes quantidades de dados não estruturados, como dados binários ou texto, que podem ser acessados de qualquer lugar do mundo.  Usando esse conector, você pode se conectar ao armazenamento de blob e carregar, obter e excluir blobs, listar blobs no contêiner, fazer instantâneos dos blobs, copiar blobs e usar um gatilho para recuperar blobs.<br/><br/>[Conectores de Aplicativo + Serviços de Dados](app-service-logic-data-connectors.md)
Tabela de Armazenamento do Azure | O serviço Tabela de Armazenamento do Azure armazena grandes quantidades de dados estruturados.  O serviço é um repositório de dados NoSQL, que é ideal para armazenar dados estruturados e não relacionais.  Esse conector se conecta a uma Tabela de Armazenamento do Azure e obter, consultar, inserir, atualizar e excluir entidades e usar um gatilho para recuperar dados.<br/><br/>[Conectores de Aplicativo + Serviços de Dados](app-service-logic-data-connectors.md)
BizTalk EDIFACT | O Aplicativo de API EDIFACT recebe e envia mensagens usando o protocolo EDIFACT em comunicações B2B.<br/><br/>[Conectores B2B](app-service-logic-b2b-connectors.md)
BizTalk X12 | O Aplicativo de API X12 recebe e envia mensagens usando o protocolo X12 em comunicações B2B.<br/><br/>[Conectores B2B](app-service-logic-b2b-connectors.md)
Codificador de Arquivo Simples do BizTalk | Oferece interoperabilidade entre dados de arquivo simples (como Excel e csv) e dados XML.  Este Aplicativo de API pode converter uma instância de arquivo simples em XML e vice-versa.<br/><br/>[Conectores de Integração do BizTalk](app-service-logic-integration-connectors.md)
Codificador JSON do BizTalk | Oferece interoperabilidade entre dados JSON e XML.  Este Aplicativo de API pode converter uma instância JSON em XML e vice-versa.<br/><br/>[Conectores de Integração do BizTalk](app-service-logic-integration-connectors.md)
Regras do BizTalk | As Regras do BizTalk definem e controlam a estrutura, a operação e a estratégia de uma organização.  Políticas de negócios podem ser atualizadas sem recompilar e reimplantar os aplicativos associados.<br/><br/>[Conectores de Integração do BizTalk](app-service-logic-integration-connectors.md)
Gerenciamento de Parceiros Comerciais do BizTalk | O Aplicativo de API de Gerenciamento de Parceiros Comerciais define e persiste relações B2B usando parceiros, contratos e esquemas e certificados usados em contratos.  Essas relações são impostas usando Aplicativos de API AS2, EDIFACT e X12.<br/><br/>[Conectores B2B](app-service-logic-b2b-connectors.md)
Serviço de Transformação do BizTalk | O Aplicativo de API de Transformação do BizTalk converte dados de um formato para outro.  Você também pode carregar um mapa existente (arquivo. trfm), exibir a representação gráfica do mapa que mostra os links entre os esquemas de origem e de destino e usar a funcionalidade de "Teste" com um conteúdo XML de entrada de exemplo.  Funções internas diferentes também estão disponíveis, incluindo manipulações de cadeia de caracteres, atribuições condicionais, expressões aritméticas, formatadores de data e hora, looping e assim por diante.<br/><br/>[Conectores de Integração do BizTalk](app-service-logic-integration-connectors.md)
Validador de XML do BizTalk | O Aplicativo de API do Validador valida dados XML com relação a esquemas XML predefinidos.  Você pode usar esquemas existentes ou gerar esquemas com base em uma instância de arquivo simples, instância JSON ou em conectores existentes.<br/><br/>[Conectores de Integração do BizTalk](app-service-logic-integration-connectors.md)
Extrator de XPath do BizTalk | O Aplicativo de API do Extrator procura e extrai dados de conteúdo XML com base em um determinado XPath.<br/><br/>[Conectores de Integração do BizTalk](app-service-logic-integration-connectors.md)
Box | O Box é um serviço de compartilhamento de arquivos.  O Conector do Box se conecta ao Box e pode carregar, obter, excluir e listar arquivos e usa um gatilho para recuperar arquivos.<br/><br/>[Conectores de Aplicativo + Serviços de Dados](app-service-logic-data-connectors.md)
Chatter | O Chatter é uma rede social empresarial.  O Conector do Chatter se conecta ao Chatter e pode postar mensagens, pesquisar e usar um gatilho para recuperar novas mensagens.<br/><br/>[Conectores sociais](app-service-logic-social-connectors.md)
DropBox | O Dropbox é um serviço de hospedagem de arquivos.  Usando este conector, você pode se conectar ao Dropbox e carregar, obter, excluir e listar arquivos e usar um gatilho para recuperar arquivos.<br/><br/>[Conectores de Aplicativo + Serviços de Dados](app-service-logic-data-connectors.md)
Facebook | O Facebook é um serviço de rede social.  O conector do Facebook se conecta ao Facebook e pode publicar mensagens, vídeos, fotos, ofertas e assim por diante.  Você também pode recuperar mensagens, comentários, eventos, feeds de usuários e diferentes informações do usuário, como livros, jogos, filmes e assim por diante.  Este conector também inclui um gatilho para recuperar novas mensagens lançadas em uma página.<br/><br/>[Conectores sociais](app-service-logic-social-connectors.md)
Arquivo  | O Conector de Arquivo permite aos clientes enviar ou carregar arquivos em um sistema de arquivos ou rede e receber ou baixar arquivos de um sistema de arquivos ou rede.  Usando este conector, você pode se conectar ao servidor de arquivos local e carregar, obter, excluir e listar arquivos e também usar um gatilho para recuperar arquivos.<br/><br/>[Conectores de protocolo](app-service-logic-protocol-connectors.md)
FTP<br/>FTPS | O FTP (File Transfer Protocol) é um protocolo de rede popular para transferência de arquivos de um host para outro.  O Conector de FTP pode carregar, obter, excluir e listar arquivos e usar um gatilho para recuperar arquivos.<br/><br/>[Conectores de protocolo](app-service-logic-protocol-connectors.md)
HTTP<br/>HTTPS | O Ouvinte de HTTP abre um ponto de extremidade que atua como servidor HTTP e ouve solicitações HTTP de entrada.<br/><br/>[Conectores de protocolo](app-service-logic-protocol-connectors.md)
HTTP<br/>HTTPS | O Conector de HTTP envia dados para servidores HTTP usando o protocolo HTTP ou HTTPS.<br/><br/>[Conectores de protocolo](app-service-logic-protocol-connectors.md)
Marketo | O Marketo desenvolve softwares de automação de marketing que fornecem marketing de entrada, marketing social, CRM e outros serviços relacionados.  O Conector do Marketo se conecta ao Marketo e pode criar/atualizar clientes potenciais, obter alterações de clientes potenciais, agendar e solicitar campanhas, obter clientes potenciais, obter informações sobre campanhas/listas, adicionar e remover clientes potenciais da lista.<br/><br/>[Conectores empresariais](app-service-logic-enterprise-connectors.md)
Microsoft Office 365 | O Conector do Office 365 pode enviar e receber emails, gerenciar seu calendário e gerenciar seus contatos usando sua conta do Office 365.  Você pode também enviar, receber e obter emails, criar e excluir eventos de seu calendário e criar, atualizar, obter e excluir seus contatos.<br/><br/>[Conectores de Aplicativo + Serviços de Dados](app-service-logic-data-connectors.md)
Microsoft OneDrive | O Microsoft OneDrive é um serviço de hospedagem de arquivos.  O Conector do OneDrive  se conecta ao seu Microsoft OneDrive pessoal e carregar, obter, excluir e listar arquivos e usar um gatilho para recuperar arquivos.<br/><br/>[Conectores de Aplicativo + Serviços de Dados](app-service-logic-data-connectors.md)
Microsoft SharePoint | O Conector do Microsoft SharePoint se conecta ao SharePoint Online ou Microsoft SharePoint Server local e gerenciar documentos e itens de lista.  Você também pode criar, atualizar, obter e excluir documentos e itens de lista.  Há suporte para diferentes métodos de autenticação, como credenciais padrão, OAuth 2.0, autenticação do Windows e autenticação baseada em formulário.  <br/><br/>[Conectores de Aplicativo + Serviços de Dados](app-service-logic-data-connectors.md)
Microsoft SQL Server | O conector do Microsoft SQL se conecta ao SQL Server local ou ao Banco de Dados SQL do Azure.  Você pode criar, atualizar, obter e excluir entradas em uma tabela de banco de dados SQL.<br/><br/>[Conectores de Aplicativo + Serviços de Dados](app-service-logic-data-connectors.md)
Microsoft Yammer | O Microsoft Yammer é uma rede social empresarial.  O conector do Yammer se conecta ao Yammer, inclui uma ação de postagem de mensagem e um gatilho para recuperar novas mensagens.<br/><br/>[Conectores sociais](app-service-logic-social-connectors.md)
MongoDB | O conector do MongoDB se conecta ao Banco de Dados do Servidor Mongo (local ou na nuvem) e cria, atualiza, obtém e exclui entradas em uma coleção do MongoDB.<br/><br/>[Conectores empresariais](app-service-logic-enterprise-connectors.md)
Banco de Dados do Oracle | O conector do Banco de Dados do Oracle se conecta ao banco de dados do Oracle local e  pode criar, atualizar, obter e excluir entradas em uma tabela de banco de dados.  <br/><br/>[Conectores de Aplicativo + Serviços de Dados](app-service-logic-data-connectors.md)
POP3<br/>IMAP | O POP3 (Post Office Protocol) é o protocolo usado por um cliente de email para recuperar emails de um servidor de email.  O Conector do POP3 se conecta a um servidor POP3 e inclui um gatilho para recuperar emails com anexos.<br/><br/>[Conectores de protocolo](app-service-logic-protocol-connectors.md)
QuickBooks | Usando o conector do QuickBooks, você pode criar, atualizar, ler, excluir e consultar entidades diferentes do Intuit QuickBooks, como clientes, itens, notas fiscais e assim por diante.<br/><br/>[Conectores empresariais](app-service-logic-enterprise-connectors.md)
Salesforce | O conector do Salesforce gerencia entidades diferentes, como contas, clientes potenciais, oportunidades, casos e assim por diante na sua conta do Salesforce.  Você também pode criar, atualizar, inserir, consultar e excluir diferentes entidades.<br/><br/>[Conectores empresariais](app-service-logic-enterprise-connectors.md)
SAP | O conector do SAP se conecta a um servidor SAP local e invoca RFCs, BAPIs e tRFCs e envia IDOCs.<br/><br/>[Conectores empresariais](app-service-logic-enterprise-connectors.md)
SFTP | O SFTP (SSH File Transfer Protocol) é um protocolo comum usado para transferir arquivos com segurança.  O conector de SFTP pode carregar, obter, excluir e listar arquivos e usa um gatilho para recuperar arquivos.<br/><br/>[Conectores de protocolo](app-service-logic-protocol-connectors.md)
SMTP | O SMTP (Simple Mail Transfer Protocol) é o protocolo usado entre um cliente e um servidor de email para enviar emails.  O Conector de SMTP se conecta a um servidor SMTP e envia emails com anexos.  <br/><br/>[Conectores de protocolo](app-service-logic-protocol-connectors.md)
SugarCRM | O conector do SugarCRM se conecta ao SugarCRM Online e pode criar, atualizar, ler e excluir diferentes tipos de módulos, como contas, contatos, produtos e assim por diante.<br/><br/>[Conectores empresariais](app-service-logic-enterprise-connectors.md)
Twilio | O Twilio é um serviço de SaaS voltado à comunicação.  O Conector do Twilio conecta-se ao Twilio e pode enviar, obter e listar mensagens, listar o uso, obter os números de telefone gratuitos, os números de celular, números locais e números de chamadas recebidas disponíveis, obter os números de chamadas recebidas e adicionar números de chamadas recebidas.<br/><br/>[Conectores sociais](app-service-logic-social-connectors.md)
Twitter | O Twitter é um serviço de rede social online que permite aos usuários enviar e ler mensagens curtas de até 140 caracteres.  O Conector do Twitter se conecta ao Twitter e pode obter a linha do tempo dos usuários, pesquisar tweets, obter seguidores, obter amigos, pesquisar usuários, obter a linha do tempo da página inicial, obter a linha do tempo de referências, postar tweets, postar tweets para um usuário e enviar mensagens diretas.  O Conector do Twitter também usa gatilhos, como Obter tweets por palavra-chave, Obter tweets por usuário e Obter tweets por hashtag.<br/><br/>[Conectores sociais](app-service-logic-social-connectors.md)
