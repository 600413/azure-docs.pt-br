# <a name="what-products-can-be-published-via-the-cloud-partner-portal"></a>Quais produtos podem ser publicados por meio do Portal de Parceiros de Nuvem

O Portal de Parceiros de Nuvem é o seu hub para publicar sua imagem de VM (máquina virtual) certificada pelo Azure no Microsoft Azure Marketplace ou seu aplicativo Cortana Intelligence/Microsoft R Solutions no Microsoft AppSource.  

As imagens de VM são imagens pré-configuradas com um sistema operacional totalmente instalado e um ou mais aplicativos. Para publicar sua imagem de máquina virtual no Azure Marketplace, é necessário cumprir o [pré-requisito técnico](https://docs.microsoft.com/azure/marketplace-publishing/marketplace-publishing-vm-image-creation-prerequisites) para certificar a sua imagem.
