# Exemplos de código do cliente de início rápido do Banco de Dados SQL


Este tópico fornece links para exemplos de código de início rápido que você pode usar para se conectar ao Banco de Dados SQL do Azure. Os exemplos abrangem:


- Diversas linguagens de programação.
- Windows, Linux e Mac OS, como os sistemas operacionais que podem ser executados em seu programa cliente.
- Exemplos de código curtos de início rápido.
- Mais exemplos que tratam erros transitórios com lógica de repetição automática.
- Exemplos de código que convertem os conjuntos de resultados relacionais em um formato orientado ao objeto.


> [AZURE.NOTE]Desde 10 de junho de 2015, os exemplos de código para outras linguagens estão sendo preparados, e os links para eles serão adicionados a este tópico.
