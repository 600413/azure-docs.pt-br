# Visão geral: ferramentas de gerenciamento para o Banco de Dados SQL

Este tópico explora e compara as ferramentas e opções de gerenciamento de Bancos de Dados SQL para que você possa escolher a ferramenta certa para o trabalho, para sua empresa e para você. Escolher a ferramenta certa depende de quantos bancos de dados você gerencia, da tarefa e da frequência de execução de uma tarefa.



## Portal do Azure


O [Portal do Azure](http://portal.azure.com) é um portal de gerenciamento baseado na Web, no qual você pode criar, atualizar, excluir servidores lógicos e Bancos de Dados SQL do Azure, além de monitorar os recursos do banco de dados. Essa ferramenta é excelente se você estiver começando a usar o Azure, se estiver gerenciando uma pequena quantidade de bancos de dados ou se precisar fazer alguma coisa rapidamente.

Para obter informações mais detalhadas sobre como usar o portal, confira [Gerenciar Bancos de Dados SQL usando o Portal do Azure](sql-database-manage-portal.md).

## SQL Server Management Studio e SQL Server Data Tools no Visual Studio


SQL Server Management Studio (SSMS) e SQL Server Data Tools (SSDT) no Visual Studio são ferramentas cliente que executam em seu computador e permitem a conexão, gerenciamento e desenvolvimento do banco de dados na nuvem. Se você for um desenvolvedor de aplicativos familiarizado com o Visual Studio ou com outros ambientes de desenvolvimento integrados (IDEs), tente usar o SSDT no Visual Studio. Se você precisar de recursos avançados de SQL que ainda não são totalmente compatíveis com o SSDT, por exemplo, gerenciamento de Bancos de Dados SQL Server em ambientes híbridos, use o SSMS.

Para saber mais sobre como gerenciar seus Bancos de Dados SQL do Azure com SSMS e SSDT, confira [Gerenciar Bancos de Dados SQL usando o SSMS](sql-database-manage-azure-ssms.md)


## Ferramentas de linha de comando

Você pode usar ferramentas de linha de comando, como o PowerShell para gerenciar Bancos de Dados SQL do Azure e automatizar implantações de recursos do Azure. A Microsoft recomenda essa ferramenta para gerenciar uma grande quantidade de Banco de Dados SQL do Azure e implantar alterações de recursos em um ambiente de produção.

Para saber mais sobre como gerenciar seus Bancos de Dados SQL do Azure com ferramentas de linha de comando, [clique aqui](sql-database-command-line-tools.md)
