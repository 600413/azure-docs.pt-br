# Tópicos sobre ferramentas de bancos de dados elásticos do Banco de Dados SQL do Azure

Os recursos de banco de dados elástico do BD SQL do Azure são projetados para simplificar o desenvolvimento e gerenciamento de camadas de dados especialmente para os desenvolvedores de Software como Serviço (SaaS), no qual é usado um grande número de bancos de dados para oferecer suporte a uma base dinâmica de clientes finais.

As **ferramentas de banco de dados elástico** ajudarão você a criar e gerenciar aplicativos que utilizam padrões de fragmentação para expansão entre vários bancos de dados. Estão incluídos os recursos de biblioteca cliente do banco de dados elástico para suporte ao gerenciamento de fragmentos, roteamento dependente de dados e consulta entre fragmentos do ADO.Net. Também está incluída a ferramenta de divisão-mesclagem, que simplifica o gerenciamento de fragmentos permitindo que você mova dados (divisão) de um fragmento para outro (mesclagem). Essas ferramentas complementam os pools de bancos de dados elásticos, que permitem preços previsíveis com características de desempenho garantido para qualquer número de bancos de dados. O recurso de trabalho associado fornece uma ferramenta fácil e confiável para atualizar e gerenciar todos os bancos de dados do pool com scripts T-SQL.
