# <a name="view-and-update-an-azure-sql-database-server-and-its-settings"></a>Exibir e atualizar um servidor de Banco de Dados SQL do Azure e suas configurações

Você pode exibir e atualizar um servidor de Banco de Dados SQL do Azure e suas configurações usando o [Portal do Azure]((sql-database-manage-portal.md), [PowerShell](sql-database-manage-powershell.md), a API REST ou C#. 

## <a name="view-and-update-an-azure-sql-server-settings-using-the-azure-portal"></a>Exibir e atualizar as configurações do Azure SQL Server usando o Portal do Azure
1. Abra a folha **Servidores SQL** no [Portal do Azure](https://portal.azure.com/). 

    ![novo sql server](./media/sql-database-get-started/new-sql-server.png)

2. Clique no servidor com o qual você deseja trabalhar e, em seguida, clique na configuração desejada na folha de servidores SQL. 

    ![folha do sql server](./media/sql-database-get-started/sql-server-blade.png)

> [!TIP]
> Para ver um tutorial de introdução ao uso do Portal do Azure, veja [Introdução aos servidores do Banco de Dados SQL do Azure, bancos de dados e regras de firewall usando o portal do Azure e o SQL Server Management Studio](sql-database-get-started.md).
>
