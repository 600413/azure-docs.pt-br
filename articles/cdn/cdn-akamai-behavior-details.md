# Azure CND dos detalhes do comportamento do Akamai

Este artigo descreve os detalhes de comportamento permitidos específicos para a **CDN do Azure do Akamai**.

> [AZURE.IMPORTANT] As informações neste artigo não se aplicam à CDN do Azurede pontos de extremidade Verizon.

## Portas de origem permitidas

A **CDN do Azure do Akamai** permite acesso a um subconjunto de portas TCP no servidor de origem. Abaixo está a lista de intervalos de porta permitidos.
