# Novidades do Aprendizado de Máquina do Azure

As atualizações de julho de 2016 do Aprendizado de Máquina (ML) do Microsoft Azure fornecem os seguintes recursos:

* Os serviços da Web agora são gerenciados como recursos do Azure gerenciados por meio das interfaces do [Azure Resource Manager](../resource-group-overview.md), permitindo os seguintes aprimoramentos:
	* Há novas [APIs REST](https://msdn.microsoft.com/library/azure/Dn950030.aspx) para implantar e gerenciar seus serviços da Web baseados no Resource Manager.
	* Há um novo portal dos [Serviços da Web do Aprendizado de Máquina do Microsoft Azure](https://services.azureml.net/) que fornece um local para gerenciar todos os aspectos do serviço da Web.
* Incorpora um novo modelo de implantação de serviço da Web de várias regiões, baseado em assinatura, usando APIs baseadas no Resource Manager que utilizam o Provedor de Recursos do Resource Manager para Serviços da Web.
* Introduz novos [planos de preços](https://azure.microsoft.com/pricing/details/machine-learning/) e recursos de gerenciamento de planos usando o novo RP do Resource Manager para Cobranças.
	* Agora você pode [implantar seu serviço da Web em várias regiões](machine-learning-how-to-deploy-to-multiple-regions.md) sem a necessidade de criar uma nova assinatura em cada região.
* Fornece [estatísticas de uso](machine-learning-manage-new-webservice.md) do serviço da Web.
* Simplifica o teste da função de RSS AM do Azure usando dados de exemplo.
* Fornece uma nova página de teste do BES com dados de exemplo e histórico de envio de trabalhos.

Além disso, o estúdio do Aprendizado de Máquina foi atualizado para permitir que você implante o novo modelo de serviço da Web ou continuar a implantar o modelo de serviço da Web clássico.
