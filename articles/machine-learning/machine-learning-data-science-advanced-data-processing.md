# Guia de aprendizagem: Processamento de dados avançado no Azure 

Este mapa percorre o processo de preparação de dados a serem usados em um experimento de Aprendizado de Máquina do Azure. Ele mostra como mover, processar e criar amostrar de dados para prepará-los para o aprendizado de máquina. O processo de preparação de dados descrito aqui é uma etapa no mapa fornecido nas [Soluções de análise avançadas de compilação no Azure](machine-learning-data-science-how-to-create-machine-learning-service.md). O guia ADAPT descreve as tarefas de ciência de dados normalmente envolvidas na criação e implantação de modelos de previsão com o Aprendizado de Máquina do Azure.
