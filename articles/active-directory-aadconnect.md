<h1 id="vnettut1">Conexão do Active Directory do Azure</h1>

O assistente de Conexão do Active Directory do Azure é a única ferramenta e experiência orientada para conectar seus diretórios locais ao Active Directory do Azure.  O assistente implanta e configura todos os componentes necessários para obter a integração e o funcionamento do seu diretório, incluindo os serviços de sincronização, a sincronização de senha ou os Serviços de Federação do Active Directory (AD FS), além dos pré-requisitos, como o módulo Azure AD PowerShell.

>[AZURE.NOTE] **A Conexão do Active Directory do Azure abrange a funcionalidade que foi lançada anteriormente como DirSync e AAD Sync. Estas ferramentas não serão lançadas individualmente.** 

> **A Conexão do Active Directory do Azure é agora a sua loja única para a sincronização, logon e todos os outros aspectos do seu local para a integração do AD do Azure.**


Se atualmente você estiver avaliando ou usando a Ferramenta de Sincronização do Active Directory do Azure (DirSync), os Serviços de Sincronização do Active Directory do Azure (AAD Sync) ou o Forefront Identity Manager 2010 R2, consulte [Ferramentas de integração de diretório](http://msdn.microsoft.com/library/azure/dn757582.aspx "Directory Integration Tools") para obter mais informações.


##  Visualização pública da Conexão do Active Directory do Azure 

A atual visualização pública da Conexão do Active Directory do Azure fornece uma experiência interativa para integrar uma ou várias florestas do Active Directory do Windows Server com o AD do Azure. 

[Baixar a visualização pública da Conexão do AD do Azure](http://connect.microsoft.com/site1164/program8612 "Azure Active Directory Connect")

Com esta versão de visualização da Conexão do Active Directory do Azure, você pode fazer o seguinte: 

- Escolher as configurações expressas para uma configuração rápida e fácil para uma floresta única em 4 cliques
- Escolher as configurações personalizadas para configurar várias florestas ou selecionar o AD FS para a experiência de logon único (SSO)
- Configurar as opções de sincronização adicionais, como o modo Híbrido do Exchange, o write-back de senha e o atributo de ID alternativo
