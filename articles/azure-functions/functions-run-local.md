# <a name="how-to-code-and-test-azure-functions-locally"></a>Como codificar e testar as funções do Azure localmente
> [!NOTE]
> No momento, a experiência de desenvolvimento local e de ferramentas do Azure Functions está em visualização e será aprimorada de forma considerável antes da versão final. A maneira mais fácil de executar o host de do Azure Functions localmente é usar a [CLI do Azure Functions](https://go.microsoft.com/fwlink/?linkid=832752) e instalá-lo do npm. No momento, apenas o Windows tem suporte. A documentação completa estará disponível em breve. Para fornecer comentários ou arquivar bugs sobre a CLI do Azure Functions, envie uma dúvida no repositório GitHub [azure-webjobs-sdk-script](https://github.com/Azure/azure-webjobs-sdk-script).
