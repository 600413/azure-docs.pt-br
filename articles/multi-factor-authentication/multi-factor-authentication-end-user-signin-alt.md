# Conectando-se usando um método alternativo com o Azure Multi-Factor Authentication


A seção a seguir mostrará como se conectar com um método alternativo quando seu método principal não estiver disponível.

## Para se conectar com um método alternativo

<ol>

<li>Entre em um aplicativo ou serviço, como o Office 365, usando seu nome de usuário e senha.</li>
<li>Selecione a opção Usar uma verificação diferente. Opções diferentes serão exibidas. O número que você vê se baseia na quantidade que foi configurada.</li>


<center>![Configuração](./media/multi-factor-authentication-end-user-signin-alt/alt.png)</center>

<li>Escolha um método alternativo e conecte-se.</li>
