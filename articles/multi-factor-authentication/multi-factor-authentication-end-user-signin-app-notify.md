# Conectando-se ao aplicativo móvel usando notificação com o Azure Multi-Factor Authentication

As informações a seguir descrevem a experiência de usar a autenticação multifator com seu aplicativo móvel quando você recebe uma notificação.

## Para se conectar com uma notificação enviada ao seu aplicativo móvel

<ol>

<li>Entre em um aplicativo ou serviço, como o Office 365, usando seu nome de usuário e senha.</li>
<li>A Microsoft enviará uma notificação.</li>


<center>![Configuração](./media/multi-factor-authentication-end-user-signin-app-notify/notify.png)</center>

<li>Atenda o telefone e pressione a tecla de verificação.</li>

<center>![Configuração](./media/multi-factor-authentication-end-user-signin-app-notify/phone.png)</center>


<li>Agora você deve estar conectado.</li>
