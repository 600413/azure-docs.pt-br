# Gerenciar as configurações do Azure Multi-Factor Authentication no portal do Azure

Se você usar a autenticação multifator com o Azure, será conveniente gerenciar as configurações da verificação de segurança adicional por meio do portal do Azure.

## Para acessar as configurações da verificação de segurança adicional no portal do Azure

<ol>
<li>Faça logon no Portal do Azure. <li>Na parte superior do portal do Azure, clique no seu nome de usuário. Isso abrirá uma caixa suspensa. <li>Na caixa suspensa, selecione Verificação de segurança adicional. Isso abrirá a seção de verificação de segurança adicional.

<center>![Configuração](./media/multi-factor-authentication-end-user-manage-azure/changesettings.png)</center>

<li>Agora, você deve estar na página de prova e pode alterar as configurações.</li>

<center>![Configuração](./media/multi-factor-authentication-end-user-manage-azure/proofup.png)</center>
