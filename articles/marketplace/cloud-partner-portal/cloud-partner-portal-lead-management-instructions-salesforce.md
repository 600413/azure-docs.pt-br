# <a name="lead-management-instructions-for-salesforce"></a>Instruções de gerenciamento de leads para Salesforce

Este documento fornece instruções sobre como configurar o sistema Salesforce para que a Microsoft possa fornecer vendas potenciais. 

1. Vá para Configuração. <br/>
![Imagem de configuração do Salesforce](./media/cloud-partner-portal-lead-management-instructions-salesforce/salesforce1.png)

2. Na barra de navegação, vá para Compilar->Personalizar->Leads->Web a Lead. <br/>
![Imagem de Web para lead do Salesforce](./media/cloud-partner-portal-lead-management-instructions-salesforce/salesforce2.png)

3. Clique no botão "Criar Formulário Web a Lead". <br/>
![Imagem do formulário de criação do Salesforce](./media/cloud-partner-portal-lead-management-instructions-salesforce/salesforce3.png)

4. Na próxima página – não se preocupe com o campo no formulário e clique em "Gerar". <br/>
![Imagem de geração do Salesforce](./media/cloud-partner-portal-lead-management-instructions-salesforce/salesforce4.png)

5. Copie o OID e cole-o no campo <b>Identificador de Objeto</b> no Portal do Cloud Partner. <br/>
![Imagem do OID de Salesforce](./media/cloud-partner-portal-lead-management-instructions-salesforce/salesforce5.png)
