# Tente o Backup do Azure em 10mins
É extremamente fácil à instalação do Backup do Azure e a seguir como-vídeos de instruções deve ajudá-lo a começar.

## Etapa 1: Obter uma assinatura do Azure
+ Se você não tiver uma assinatura do Azure, você pode começar com uma oferta de avaliação gratuita que permite acessar qualquer serviço do Azure. Visite o link a seguir [http://azure.microsoft.com/en-in/pricing/free-trial/](http://azure.microsoft.com/en-in/pricing/free-trial) para obter mais informações.
+ Você poderá ignorar esta etapa, você tem uma assinatura do Azure.

## Etapa 2: Configurar o serviço de Backup do Azure
Explique as etapas mencionadas no vídeo nos [http://azure.microsoft.com/en-in/documentation/videos/getting-started-with-azure-backup-1-of-3-set-up-a-backup-vault-on-azure/](http://azure.microsoft.com/en-in/documentation/videos/getting-started-with-azure-backup-1-of-3-set-up-a-backup-vault-on-azure/) para:

+	Criar um cofre de backup

+	Baixar as "credenciais do cofre"

+	Baixar o Azure Backup agent

> [AZURE.VIDEO getting-started-with-azure-backup-1-of-3-set-up-a-backup-vault-on-azure]

## Etapa 3: Preparar sua máquina local
Explique as etapas mencionadas no vídeo nos [http://azure.microsoft.com/en-in/documentation/videos/getting-started-with-azure-backup-2-of-3-prepare-your-production-server-for-azure-backup/](http://azure.microsoft.com/en-in/documentation/videos/getting-started-with-azure-backup-2-of-3-prepare-your-production-server-for-azure-backup/) para:

+ Instale o Azure Backup agent no servidor DPM ou servidor/cliente do Windows.

+ Registre-se a máquina com o Cofre de backup


> [AZURE.VIDEO getting-started-with-azure-backup-2-of-3-prepare-your-production-server-for-azure-backup]

## Etapa 4: Backup e restauração de dados
Explique as etapas mencionadas no vídeo nos [http://azure.microsoft.com/en-in/documentation/videos/getting-started-with-azure-backup-3-of-3-start-backing-up-your-production-server/](http://azure.microsoft.com/en-in/documentation/videos/getting-started-with-azure-backup-3-of-3-start-backing-up-your-production-server/) para:

+ Dados de backup de um servidor

+ Restaurar dados de backup do Azure

> [AZURE.VIDEO getting-started-with-azure-backup-3-of-3-start-backing-up-your-production-server]
