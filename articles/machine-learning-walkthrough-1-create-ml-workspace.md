# Etapa 1: Criar um espaço de trabalho de Aprendizado de Máquina do Azure

Para usar o Estúdio AM, é necessário ter um espaço de trabalho de AM. Esse espaço de trabalho contém as ferramentas necessárias para criar, gerenciar e publicar testes.  

## Para criar um espaço de trabalho  

1.	Entre na sua conta Microsoft Azure.
2.	No painel de serviços do Microsoft Azure, clique em **APRENDIZADO DE MÁQUINA**.  
![Create workspace][1]

3.	Clique em **CRIAR UM ESPAÇO DE TRABALHO DE AM**.
4.	Na página **CRIAÇÃO RÁPIDA**, insira suas informações de espaço de trabalho e clique em **CRIAR UM ESPAÇO DE TRABALHO DE AM**.

	>**OBSERVAÇÃO** - O **PROPRIETÁRIO DO ESPAÇO DE TRABALHO** é sua conta da Microsoft (por exemplo, name@outlook.com).

Após seu espaço de trabalho AM ser criado, você o verá listado na página **aprendizado de máquina**.  

>**Dica** - Você pode compartilhar os testes em que está trabalhando convidando outras pessoas para seu espaço de trabalho. Pode fazer isso no Estúdio AM na página **CONFIGURAÇÕES**. Basta possui a conta da Microsoft ou a conta da empresa de cada usuário.
