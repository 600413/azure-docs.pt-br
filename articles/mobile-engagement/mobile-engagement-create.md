# <a name="create-an-azure-mobile-engagement-app"></a>Criar um Aplicativo do Azure Mobile Engagement
Este artigo mostra como usar o método **Criação Rápida** para criar um novo Aplicativo do **Engajamento Móvel do Azure**. O artigo também mostra como navegar até o portal do **Mobile Engagement** para iniciar o monitoramento e o gerenciamento de seus aplicativos. 

Observe que você deve adicionar um conjunto mínimo de "integração básica" para poder coletar dados para seu aplicativo e enviar notificações por push. A documentação de integração completa pode ser encontrada na [integração do Mobile Engagement](mobile-engagement-windows-store-integrate-engagement.md)

> [!IMPORTANT]
> Para concluir qualquer tutorial do Mobile Engagement do Azure, você precisará ter uma conta ativa do Azure. Se você não tiver uma conta, poderá criar uma conta de avaliação gratuita em apenas alguns minutos. Para obter detalhes, consulte <a href="http://azure.microsoft.com/pricing/free-trial/?WT.mc_id=A0E0E5C02&amp;returnurl=http%3A%2F%2Fwww.windowsazure.com%2Fen-us%2Fdevelop%2Fmobile%2Ftutorials%2Fget-started%2F" target="_blank">Avaliação Gratuita do Azure</a>.
>

## <a name="navigate-to-your-mobile-engagement-portal"></a>Navegue até o portal do Mobile Engagement
Para começar a monitorar e gerenciar seu aplicativo, navegue até o portal do Engajamento Móvel clicando no botão **Portal Engagement** na barra superior.

Quando você estiver no portal do Engajamento Móvel, poderá analisar, criar e gerenciar os segmentos, acessar os usuários etc.:    

* [Monitorar dados em tempo real sobre o seu aplicativo](mobile-engagement-user-interface-monitor.md)
* [Analisar dados históricos sobre seu aplicativo](mobile-engagement-user-interface-analytics.md)
* [Criar e gerenciar segmentos de usuários para identificar padrões de uso](mobile-engagement-user-interface-segments.md)
* [Entrar em contato com os usuários do seu aplicativo com notificações por push](mobile-engagement-user-interface-reach.md)
