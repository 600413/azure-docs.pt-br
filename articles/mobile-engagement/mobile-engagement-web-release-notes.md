#Notas de versão do SDK para Web do Azure Mobile Engagement

##2\.0.1 (10/06/2016)

-   SDK desabilitado no IE8 e IE9.
-   Detecção corrigida do navegador da Web Opera.

##2\.0.0 (07/06/2016)

-   Versão Inicial do Mobile Engagement do Azure.
-   A configuração do appId é substituída por uma configuração de cadeia de conexão.
-   APIs adicionadas para habilitar/desabilitar o Agente.
-   Aprimoramentos de segurança.
-   Use APIs nativas do JSON em navegadores.
-   Removida a API para enviar e receber mensagens entre dispositivos.
-   Retornos de chamada removidos relacionados à conexão XMPP substituída. 
-   Esta versão não é compatível com o recurso Reach.
