# <a name="commands"></a>Comandos
|||
|---|---|
|az mysql server|Gerencia servidores. Servidores contêm bancos de dados, data warehouses e pools elásticos.|
|az mysql server create|Cria um servidor MySQL.|
|az mysql server delete|Exclui um servidor MySQL.|
|az mysql server list|Obtém uma lista de todos os servidores MySQL em um grupo de recursos.|
|az mysql server show|Mostra detalhes de um servidor MySQL.|
|az mysql server update|Atualiza um servidor.|
|az mysql server restore|Restaura um servidor para um ponto anterior no tempo.|
|az mysql server firewall-rule|Gerencia regras de firewall do servidor.|
|az mysql server firewall-rule create|Cria uma regra de firewall.|
|az mysql server firewall-rule delete|Exclui uma regra de firewall.|
|az mysql server firewall-rule list|Lista as regras de firewall.|
|az mysql server firewall-rule show|Mostra os detalhes de uma regra de firewall.|
|az mysql server firewall-rule update|Atualiza uma regra de firewall.|
|az mysql db|Gerencia bancos de dados.|
|az mysql db list|Lista todos os bancos de dados MySQL em um servidor.|
|az mysql server-logs list|Lista arquivos de log para um servidor.|
|az mysql server-logs download|Baixa logs do servidor para o local.|
|az mysql server configuration list|Lista os parâmetros de configuração do servidor.|
|az mysql server configuration show|Mostra detalhes do parâmetro de configuração do servidor.|
|az mysql server configuration update|Modifica o valor do parâmetro de configuração do servidor.|
|||
