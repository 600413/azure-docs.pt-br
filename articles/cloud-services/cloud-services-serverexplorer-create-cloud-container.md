# Como usar o Visual Studio para criar um contêiner de Serviço de Nuvem

Para exibir o **Gerenciador de Servidores** no Visual Studio, na barra de menus, escolha **Exibir**, **Gerenciador de Servidores**.

1.  Expanda o nó **Azure**

2.  Se o usuário não estiver conectado, uma caixa de diálogo de entrar será automaticamente solicitada. Siga as instruções para fazer logon para a assinatura do Azure

3.  Clique no nó **Serviço de Nuvem** e escolha **Criar Serviço de Nuvem…**

4.  Uma caixa de diálogo **Criar Serviço de Nuvem** é exibida. Preencha os campos **Assinatura**, **Nome**, **Região ou Grupo de Afinidade** e clique no botão **Criar** para criar um contêiner de Serviço de Nuvem vazio
