# Galeria de Imagens do Governo do Microsoft Azure
<p> A Galeria de Imagens do Governo do Microsoft Azure é uma experiência semelhante ao portal do Azure público. Os clientes pode selecionar implantar imagens pré-integradas da Microsoft e nossos parceiros ou carregar suas próprias VHDs. Isso fornece flexibilidade para implantar suas próprias imagens padronizadas, se necessário.

A seguir está uma lista de imagens disponíveis na galeria de imagens do Governo do Azure. Algumas imagens predefinidas incluem licenciamento pré-pago para software específico. Examine a página <a href="http://azure.microsoft.com/pricing/details/virtual-machines/">Preços da Máquina Virtual</a> para obter mais orientações e trabalhar com sua equipe de conta da Microsoft ou revendedor para obter preços específicos do Governo do Azure.
