# Folha de consulta de cmdlets do PowerShell do Microsoft Azure HDInsight para tarefas comuns do Hadoop

O Microsoft Azure HDInsight fornece cmdlets do PowerShell para executar tarefas do Hadoop em um cluster. A **Folha de Consulta de Cmdlets do PowerShell do Microsoft Azure HDInsight** foi desenvolvida para fornecer a você cmdlets do PowerShell para as tarefas mais comuns do Hadoop, como criar um cluster, executar trabalhos, excluir um cluster, etc. A folha de consulta também fornece uma lista de todos os cmdlets do PowerShell disponíveis para o Microsoft Azure HDInsight.

## Folha de consulta para HDInsight: modelo de Gerenciamento de Serviços

**Baixe aqui a folha de consulta: [Folha de Consulta de Cmdlets do PowerShell do Microsoft Azure HDInsight](http://download.microsoft.com/download/B/7/D/B7DBB509-164D-4343-9894-12D1FB053776/HDI_PowerShell_Cmdlet_CheatSheet.pdf)**
