# Introdução ao ecossistema Hadoop com uma área restrita do Hadoop

Você pode configurar uma área restrita do Hadoop (do Hortonworks) em uma máquina virtual para saber mais sobre o ecossistema do Hadoop. A área restrita fornece um ambiente de desenvolvimento local para entender o Hadoop, o HDFS (Sistema de Arquivos Distribuído Hadoop), o envio de trabalhos, etc. Para começar com uma área restrita do Hortonworks Hadoop, confira [Hortonworks Sandox](http://hortonworks.com/downloads/#sandbox) (Área restrita do Hortonworks) e examine a seção **Hortonworks Sandbox on a VM** (Hortonworks seguro em uma VM). Recomendamos que você percorra os tutoriais disponíveis com a área restrita para obter um entendimento sobre o HDFS, como os trabalhos são enviados para um cluster, como controlar trabalhos em execução em um cluster, etc.

Quando estiver familiarizado com o Hadoop, você poderá começar a usar o Hadoop no Azure, criando um cluster do HDInsight. Para obter mais informações sobre como começar, confira [Introdução ao Hadoop no HDInsight](hdinsight-hadoop-linux-tutorial-get-started.md).
