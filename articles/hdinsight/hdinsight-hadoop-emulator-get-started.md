# Introdução ao ecossistema Hadoop com uma área restrita do Hadoop em uma máquina virtual

Você pode configurar uma área restrita do Hadoop da Hortonworks em uma máquina virtual para saber mais sobre o ecossistema do Hadoop. A área restrita fornece um ambiente de desenvolvimento local para saber mais sobre o Hadoop, o HDFS (Sistema de Arquivos Distribuído Hadoop) e o envio de trabalhos.

Para começar com uma área restrita do Hadoop da Hortonworks, confira [Hortonworks Sandbox](http://hortonworks.com/downloads/#sandbox) e examine a seção **Hortonworks Sandbox on a VM**. É recomendável percorrer os tutoriais disponíveis com a área restrita para obter um entendimento sobre o HDFS, como os trabalhos são enviados para um cluster, como controlar trabalhos em execução em um cluster, etc.

Quando estiver familiarizado com o Hadoop, você poderá começar a usar o Hadoop no Azure, criando um cluster do HDInsight. Para saber mais sobre como começar, confira [Introdução ao Hadoop no HDInsight](hdinsight-hadoop-linux-tutorial-get-started.md).
