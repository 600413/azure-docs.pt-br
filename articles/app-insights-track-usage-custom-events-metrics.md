# Controlar o uso com métricas e eventos personalizados

### Em breve

O SDK do Application Insights permite que você insira linhas de código em seu aplicativo para monitorar as atividades dos usuários e, dessa forma, ajustar o aplicativo às necessidades deles.

No momento, ainda estamos passando este recurso para o novo portal do Application Insights no Microsoft Azure. (Se quiser, você pode [ver como funciona no portal antigo](http://msdn.microsoft.com/library/dn481100.aspx).)

No entanto, o que você pode fazer agora é [coletar e pesquisar logs de eventos de diagnóstico gerados com Trace, NLog ou Log4Net][diagnóstico].
