# Instalar o Visual Studio 2015

Para o desenvolvimento, a equipe do SQL Data Warehouse do Azure recomenda usar o Visual Studio 2013 ou posterior combinado ao SSDT (SQL Server Data Tools). Veja a seguir como baixar e atualizar o Visual Studio 2015, caso você ainda não tenha uma versão viável do Visual Studio instalada. As etapas a seguir o ajudarão a baixar e instalar os pré-requisitos para trabalhar com o SQL Data Warehouse do Azure.

+ [Baixar e instalar o Visual Studio 2015](#Download-and-Install-Visual-Studio-2015)
+ [Baixar e instalar a Visualização do SSDT (SQL Server Data Tools)] (#Download-and-Install-SQL-Server-Data-Tools-(SSDT)-Preview)
 
> [AZURE.NOTE]A visualização do Visual Studio 2013 e SSDT (SQL Server Data Tools) também pode ser usada com o SQL Data Warehouse do Azure.
 
##Baixar e instalar o Visual Studio 2015

Para instalar o Visual Studio 2015, siga as etapas abaixo

1. [Baixe o Visual Studio 2015](https://www.visualstudio.com/downloads) do Visual Studio online.
2. Execute o guia [Instalando o Visual Studio](https://msdn.microsoft.com/library/e2h7fzkw.aspx) do MSDN, escolhendo as configurações padrão 

##Baixar e instalar a Visualização do SSDT (SQL Server Data Tools)

O SQL Data Warehouse do Azure utiliza a última versão de visualização do SSDT (SQL Server Data Tools) disponível na

1. [Baixe a Visualização do SQL Server Data Tools](https://msdn.microsoft.com/mt429383.aspx) para Visual Studio 2013 ou 2015.
2. Siga as instruções de instalação para instalar as ferramentas de dados.
