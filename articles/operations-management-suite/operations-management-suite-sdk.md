# <a name="operations-management-suite-(oms)-sdk"></a>SDK do Operations Management Suite (OMS)
O [Microsoft Operations Management Suite (OMS)](../operations-management-suite/operations-management-suite-overview.md) é a solução de gerenciamento de TI baseada em nuvem da Microsoft que ajuda você a gerenciar e proteger sua infraestrutura local e de nuvem.  Este artigo lista a documentação e outros recursos disponíveis para acessar os serviços do OMS fora dos portais do OMS e do Azure.  Isso inclui a API REST para acesso de várias interfaces de programação e mecanismos de script, como o PowerShell.
