# Segurança do Serviço de Aplicativo do Azure

O Serviço de Aplicativo do Azure está em conformidade com os padrões essenciais do setor em relação a segurança e confiabilidade, para fornecer a nossos clientes uma plataforma segura. Temos vários recursos fáceis de usar para ajudar a proteger o tráfego de entrada e saída do seu aplicativo. Também ajudamos clientes a proteger seu código de aplicativo, fornecendo funcionalidade externa para verificar vulnerabilidades em seu aplicativo Web.
