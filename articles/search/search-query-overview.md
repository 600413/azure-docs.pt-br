# Consultas na Pesquisa do Azure
> [AZURE.SELECTOR]
- [Overview](search-query-overview.md)
- [Search Explorer](search-explorer.md)
- [Fiddler](search-fiddler.md)
- [.NET](search-query-dotnet.md)
- [REST](search-query-rest-api.md)

As soluções personalizadas que integram a Pesquisa do Azure para uma experiência de pesquisa interna precisam incluir o código que constrói uma consulta de pesquisa e a envia ao mecanismo de pesquisa para processamento.

Você pode usar a API REST ou o SDK do .NET para escrever métodos para a execução da consulta. Para testes preliminares ou exploração, você pode usar ferramentas como o Gerenciador de Pesquisa interno no Portal do Azure para consultar um índice ou usar o Fiddler para emitir qualquer solicitação válida para o serviço de pesquisa.

Visite [Vídeos, tutoriais, demonstrações e exemplos de código](search-video-demo-tutorial-list.md) para obter uma lista de exemplos e explicações passo a passo que demonstram como construir consultas no código. Quase todos os exemplos que fornecemos incluem código que consulta um serviço de pesquisa.
