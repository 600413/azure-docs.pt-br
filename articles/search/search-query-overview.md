# Consultas na Pesquisa do Azure
> [AZURE.SELECTOR]
- [Overview](search-query-overview.md)
- [Fiddler](search-fiddler.md)
- [Postman](search-chrome-postman.md)
- [.NET](search-query-dotnet.md)
- [REST](search-query-rest-api.md)

As soluções personalizadas que usam a pesquisa do Azure para uma experiência de pesquisa interna precisam incluir o código que constrói a consulta de pesquisa e a envia para um serviço de pesquisa para processamento.

Você pode usar a API REST ou o SDK do .NET para escrever métodos para a execução da consulta. Para testes ou exploração preliminares, você pode usar ferramentas como o Fiddler ou o Chrome Postman para enviar consultas ao serviço.
