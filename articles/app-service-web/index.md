<div class="content">
    <h1>Documentação de Aplicativos Web</h1>
    <p style="padding-bottom: 0px; max-width: 1050px;">Os Aplicativos Web do Azure permitem criar e hospedar sites na linguagem de programação de sua escolha, sem gerenciamento de infra-estrutura. Eles oferecem o dimensionamento automático e alta disponibilidade, compatível com Windows e Linux (visualização) e permite implantações automatizadas do GitHub, Serviços de Equipe ou qualquer repositório Git. Saiba como usar os Aplicativos Web do Azure com nossos guias de início rápido, tutoriais e exemplos.</p>
