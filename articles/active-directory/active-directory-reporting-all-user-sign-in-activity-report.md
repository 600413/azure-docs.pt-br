# Relatório de atividade de entrada de todos os usuários do Active Directory do Azure

*Esta documentação é parte do [Guia de Relatórios do Active Directory do Azure](active-directory-reporting-guide.md).*

 >[AZURE.IMPORTANT]> - O Relatório de atividade de entrada de todos os usuários é um relatório premium no modo de visualização privada. Para obter acesso ao relatório, envie um email para aadreportinghelp@microsoft.com com seu nome de domínio (algo como **contoso.onmicrosoft.com**)


O Relatório de atividade de entrada de todos os usuários contém todas as entradas de todos os usuários em seu diretório, incluindo o endereço IP, data e hora, UPN e o aplicativo ao qual o usuário se conectou.


 >[AZURE.IMPORTANT]> - O Relatório de atividade de entrada de todos os usuários está disponível atualmente apenas por meio da API de Relatório. Visite [Introdução à API de Relatório](active-directory-reporting-api-getting-started.md) para obter detalhes.
