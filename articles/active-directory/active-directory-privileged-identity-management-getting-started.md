# Introdução ao Azure AD Privileged Identity Management

## Iniciando a extensão Privileged Identity Management

1. Crie um diretório no Active Directory do Azure.
2. Adicione usuários a seu novo diretório, incluindo alguns administradores globais.

    >[AZURE.NOTE] Configure os usuários com suas contas corporativas ou de estudante. Atualmente as contas da Microsoft não podem se inscrever no Azure AD PIM.

3. Entre no [portal do Azure](https://portal.azure.com/) como um administrador global do diretório.
4. Clique em seu nome de usuário no canto superior direito do portal do Azure.
5. Selecione o diretório que você acabou de criar.
6. Clique no ícone **Novo** no painel de navegação esquerdo.
7. Selecione **Segurança + Identidade** no menu Criar.
8. Selecione **Azure AD Privileged Identity Management**.
9. Deixe a opção **Fixar no painel** marcada e clique no botão **Criar**. O Painel Privileged Identity Management será aberto.
