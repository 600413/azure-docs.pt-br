# O que fazer se você alterar o registrador DNS para o nome de domínio personalizado

Se usa somente seu nome de domínio personalizado com o AD do Azure, você pode continuar usando seu nome de domínio personalizado sem interrupção e sem outras tarefas de configuração no AD do Azure.

## Se você usar outros serviços

Se você usa o nome de domínio personalizado com o Office 365, o Intune ou outros serviços que dependem de nomes de domínio personalizado, consulte a documentação desses serviços.

-   [Como usar domínios personalizados com o Office 365](https://support.office.com/article/Add-your-users-and-domain-to-Office-365-6383f56d-3d09-4dcb-9b41-b5f5a5efd611?ui=pt-BR&rs=pt-BR&ad=US)

-   [Como usar domínios personalizados com o Intune](https://technet.microsoft.com/library/dn646966.aspx#BKMK_DomainNames)
