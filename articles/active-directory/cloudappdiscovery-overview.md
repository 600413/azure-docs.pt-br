# <a name="cloud-app-discovery-enhancements-in-azure-active-directory"></a>Melhorias do Cloud App Discovery no Azure Active Directory 
Mais de 80% de funcionários admitem usar aplicativos de SaaS não aprovados para fins de trabalho. Agora integramos o Cloud App Discovery ao Azure AD com os dados do Microsoft Cloud App Security e recursos de análise para que você possa identificar aplicativos de nuvem não gerenciados mais facilmente. Essa nova integração requer etapas de instalação diferentes das requeridas anteriormente.

## <a name="what-can-i-do-now-with-the-improvements-to-cloud-app-discovery-in-azure-ad"></a>O que posso fazer com os aprimoramentos para o Cloud App Discovery no Azure AD?

* **Maior visibilidade no uso de aplicativos de nuvem** O Cloud App Discovery no Azure AD agora descobre **mais de 15.000 aplicativos** de todo tráfego de rede da organização e de qualquer dispositivo. 
* **Sem necessidade de agentes** Essa nova versão do Cloud App Discovery não requer que agentes sejam instalados em dispositivos do usuário. Em vez disso, a descoberta é realizada com base nos arquivos de log importados de seus firewalls e proxies. Você pode descobrir aplicativos em todo tráfego de rede da organização, independentemente do dispositivo ou do sistema operacional.
* **Alertas e análise em andamento** O Cloud App Discovery no Azure AD agora fornece análise de risco contínua detalhada, além de fornecer alertas quando há um novo aplicativo em uso. Agora você pode obter conhecimento mais aprofundado sobre o uso dos aplicativos de nuvem em sua organização, assim como informações sobre o tráfego de entrada, o tráfego de saída e os principais usuários dos aplicativos descobertos.

Se você não tem o Active Directory Premium, consulte [Preço do Azure Active Directory](https://azure.microsoft.com/pricing/details/active-directory/) para obter mais detalhes.

Use este link para explorar a [nova experiência do Cloud App Discovery no Azure AD](https://portal.cloudappsecurity.com).
