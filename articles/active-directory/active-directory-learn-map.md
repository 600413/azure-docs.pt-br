# Saiba mais sobre o AD do Azure

### Outros artigos sobre este tópico
[O que é o AD do Azure?](active-directory-whatis.md)<br> [Como ele funciona?](active-directory-works.md)<br> [Introdução](active-directory-get-started.md)<br> [Próximas etapas](active-directory-next-steps.md)<br> [Saiba mais](active-directory-learn-map.md)

Já tenho uma assinatura do Azure. Como o AD do Azure está relacionado a isso? [Como as assinaturas do Azure são associadas a um diretório](active-directory-how-subscriptions-associated-directory.md)

Para tarefas administrativas do AD do Azure, como gerenciamento de usuários, gerenciamento de domínio ou configuração de logon único, use os cmdlets do módulo Active Directory do Azure para o Windows PowerShell. [Gerenciar o AD do Azure usando o Windows PowerShell](https://msdn.microsoft.com/library/azure/jj151815.aspx)

Use a Graph API do Active Directory do Azure para acessar programaticamente o AD do Azure por meio de pontos de extremidade da API REST. [Graph API do AD do Azure](https://msdn.microsoft.com/library/azure/hh974476.aspx)

Você pode adicionar seu próprio nome de domínio personalizado para o Active Directory do Azure para usá-lo com os Serviços Online da Microsoft. [Gerenciamento de domínio de Internet](active-directory-add-domain.md)

Para entender as restrições de uso e outros limites de serviço para o serviço Active Directory do Azure, consulte [Limites e restrições de serviço do AD do Azure](active-directory-service-limits-restrictions.md)
