# Azure AD Connect Sync: personalizando opções de sincronização

Os serviços de sincronização do Azure Active Directory Connect (Azure AD Connect Sync) é um componente principal do Azure AD Connect que cuida de todas as operações relacionadas à sincronização de dados de identidade entre o ambiente local e o AD do Azure na nuvem. De uma perspectiva de sincronização, o Azure AD Connect Sync é o sucessor do DirSync, Azure AD Sync e o Forefront Identity Manager com o Active Directory Connector configurado.
