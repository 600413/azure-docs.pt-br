# <a name="azure-active-directory-documentation"></a>Documentação do Azure Active Directory

O Azure Active Directory (Azure AD) é o serviço de gerenciamento de identidade e diretório multilocatário com base em nuvem da Microsoft.  O Azure AD oferece recursos de acesso e identidade para aplicativos em execução no Azure, bem como locais.
