# Perguntas frequentes sobre acesso condicional

**Posso usar as condições de dispositivo no AD do Azure se estiver usando o gerenciamento de dispositivos móveis para o Office 365?**
 
Com o MDM para O365, todas as políticas de dispositivo são mantidas como parte do Office 365. Para evitar políticas conflitantes entre o que é definido no Office 365 e o que é definido no AD do Azure, estamos atualmente impedindo a configuração dessas políticas no AD do Azure se você estiver usando o MDM para O365.
