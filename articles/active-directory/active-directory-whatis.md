# O que é o Active Directory do Azure?


###Outros artigos sobre este tópico
[O que é o AD do Azure?](active-directory-whatis.md)<br> [Como ele funciona?](active-directory-works.md)<br> [Introdução](active-directory-get-started.md)<br> [Próximas etapas](active-directory-next-steps.md)<br> [Saiba mais](active-directory-learn-map.md)

O Active Directory do Azure (AD do Azure) fornece uma maneira fácil para a sua empresa gerenciar identidades e acesso, tanto na nuvem quanto local. Os usuários podem usar uma conta corporativa ou escolar para logon único em qualquer aplicativo Web local e na nuvem, usando seu dispositivo favorito, incluindo iOS, Android e Mac OS X e dispositivos do Windows. Sua organização pode proteger aplicativos e dados confidenciais localmente e na nuvem com o Multi-Factor Authentication integrado, garantindo acesso remoto e local seguro. Ou estenda seus diretórios locais para que os operadores de informações possam usar uma única conta organizacional para acessar de maneira segura e consistente seus recursos corporativos. O Azure AD também oferece recursos de autoatendimento, relatórios e análises para reduzir custos e aumentar a segurança. E com escala corporativa e o SLA do Azure AD, sua empresa continuará em funcionamento sempre.

![][1]


> [AZURE.NOTE]Para usar o Active Directory do Azure, você precisa de uma conta do Azure. Se você ainda não tem uma conta, você pode [inscrever-se para uma conta gratuita do Azure](http://azure.microsoft.com/pricing/free-trial/).
