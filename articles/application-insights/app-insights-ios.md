# Análise de aplicativos iOS

A Microsoft oferece duas soluções para devOps de dispositivo: o [HockeyApp](http://hockeyapp.net/), para análise de fluxo de trabalho e de falhas de devOps e o [Application Insights](app-insights-overview.md), para análise de uso e de falhas.

O [HockeyApp](http://hockeyapp.net/) é a nossa solução de DevOps Móvel para a criação de aplicativos de dispositivos iOS, OS X, Android ou Windows, bem como aplicativos de plataforma cruzada baseados no Xamarin, no Cordova e no Unity. Com ele, você pode distribuir compilações para testadores beta, coletar dados de falha e obter comentários dos usuários. Ele é integrado ao Visual Studio Team Services, permitindo a compilação fácil de implantações e a integração de itens de trabalho. Saiba mais na [Base de Dados de Conhecimento do HockeyApp](http://support.hockeyapp.net/kb) e mantenha-se atualizado consultando o [Blog do HockeyApp](http://hockeyapp.net/blog/).


Use o [SDK do Application Insights para iOS](https://github.com/Microsoft/ApplicationInsights-iOS) para análise de uso. Se seu aplicativo tiver um lado de servidor, você poderá enviar os dados de cliente e de servidor para o mesmo recurso do Application Insights.


* [Introdução ao Application Insights para iOS](https://github.com/Microsoft/ApplicationInsights-iOS).
* [Introdução ao HockeyApp para Android](http://support.hockeyapp.net/kb/client-integration-ios-mac-os-x/hockeyapp-for-ios).
