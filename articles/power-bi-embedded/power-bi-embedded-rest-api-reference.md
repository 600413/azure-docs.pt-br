# API REST do Power BI Embedded

Com o **Microsoft Power BI Embedded**, você pode usar uma API REST para desenvolver programaticamente aplicativos que integrem os **Relatórios do Power BI** aos seus aplicativos.

Para saber mais sobre a API REST do **Power BI Embedded**, confira [Operações REST do Power BI](https://msdn.microsoft.com/library/mt711493).
