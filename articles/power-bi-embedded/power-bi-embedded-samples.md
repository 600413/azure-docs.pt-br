# Exemplos

[Baixe o exemplo de visualização pública do Power BI Embedded](http://go.microsoft.com/fwlink/?LinkID=761493)

Para ajudar a começar, criamos um aplicativo Web de exemplo. Você pode baixá-lo e ler o artigo [Introdução ao exemplo](power-bi-embedded-get-started-sample.md) complementar. O exemplo mostra a você como:

  -	Provisionar conteúdo
      - Criar um espaço de trabalho
      - Importar um arquivo PBIX
      - Atualizar cadeias de conexão e definir suas credenciais para os conjuntos de dados

  - Inserir um relatório com segurança
