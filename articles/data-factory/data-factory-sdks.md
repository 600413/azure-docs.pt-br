# Referência do desenvolvedor da Azure Data Factory

Você pode criar, monitorar e gerenciar as factories usando o Portal Clássico do Azure, o Azure PowerShell, a Biblioteca de Classes do .NET ou a API REST.

Método | Local do recurso | Referências para o desenvolvedor
---------------------------------------------------- | ------------------------------ | -----------
Portal do Azure | [https://portal.azure.com/](https://portal.azure.com) | [Introdução ao Azure Data Factory (Portal do Azure)](data-factory-build-your-first-pipeline-using-editor.md)
Azure PowerShell | Baixe o [Azure PowerShell](http://go.microsoft.com/?linkid=9811175&clcid=0x409) mais recente | [Referência de cmdlet](https://msdn.microsoft.com/library/dn820234.aspx) 
Biblioteca de classes do .NET | O SDK .NET do Azure Data Factory permite que você crie, monitore e gerencie as data factories do Azure e estenda o Data Factory usando uma atividade .NET. Consulte os artigos [Usar atividades personalizadas em um pipeline do Azure Data Factory](data-factory-use-custom-activities.md) e [Criar, monitorar e gerenciar data factories do Azure usando o SDK .NET do Data Factory](data-factory-create-data-factories-programmatically.md) para ajudá-lo a começar.<br/><br/><b>Baixando o Nuget mais recente</b><br/>Você pode baixar o pacote Nuget mais recente da Biblioteca de Gerenciamento do Azure Data Factory em: [https://www.nuget.org/packages/Microsoft.Azure.Management.DataFactories/](https://www.nuget.org/packages/Microsoft.Azure.Management.DataFactories/)<br/><br/>**Usando o Package Manager Console no Visual Studio**<br/>Você pode executar o seguinte comando no Package Manager Console do Visual Studio para obter a Biblioteca de gerenciamento do Azure Data Factory mais recente<br/><br/>Install-Package Microsoft.Azure.Management.DataFactories | [Referência do SDK .NET](https://msdn.microsoft.com/library/mt415893.aspx)
API REST | Você pode usar a API REST do Data Factory para criar, monitorar e gerenciar data factories do Azure. | [referência da API REST (a página pode estar em inglês)](https://msdn.microsoft.com/library/dn906738.aspx)
