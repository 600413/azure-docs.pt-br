> [AZURE.NOTE]<a name="note"></a>Você precisa de uma conta Azure para concluir este tutorial:
  >
  > + É possível [abrir uma conta Azure gratuitamente](/pricing/free-trial/?WT.mc_id=A261C142F): Você obtém créditos que podem ser usados para experimentar serviços pagos do Azure e, mesmo após eles serem utilizados, você pode manter a conta e utilizar os serviços gratuitos do Azure, como os Sites. Seu cartão de crédito nunca será cobrado, a menos que você altere explicitamente suas configurações e peça para ser cobrado. 
  >
  > + Você pode [ativar os benefícios de assinante do MSDN](/pricing/member-offers/msdn-benefits-details/?WT.mc_id=A261C142F): sua assinatura do MSDN lhe fornece créditos a cada mês, que você pode usar para serviços do Azure pagos.
  >
  >Se você desejar começar a usar o Serviço de Aplicativo do Azure antes de se inscrever em uma conta do Azure, vá para [Experimentar Aplicativo de Serviço](http://go.microsoft.com/fwlink/?LinkId=523751), e você poderá criar imediatamente um aplicativo Web iniciador de curta duração no Aplicativo de Serviço. Nenhum cartão de crédito é exigido, sem compromissos.
