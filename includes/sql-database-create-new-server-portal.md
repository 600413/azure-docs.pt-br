## <a name="create-an-azure-sql-database-logical-server"></a>Criar um servidor lógico do Banco de Dados SQL do Azure
Use as etapas a seguir no portal do Azure para criar um servidor lógico do Banco de Dados SQL.

1. Se você não estiver conectado, conecte-se ao [Portal do Azure](http://portal.azure.com).
2. Clique em **Novo**, clique em **Dados + Armazenamento** e localize o **Banco de Dados SQL**.
3. Clique em **Banco de Dados SQL** para abrir a folha Banco de Dados SQL.
4. Forneça os valores para as seguintes propriedades de servidor:
   
   * Nome do servidor
   * Logon de administrador do servidor
   * Senha
   * Assinatura (somente se você tiver várias assinaturas)
   * Grupo de recursos (novo ou existente)
   * Local
5. Clique em **Criar**. Veja na área de notificação que a implantação foi iniciada.
6. Aguarde até que a implantação seja concluída antes de prosseguir para a próxima etapa.
