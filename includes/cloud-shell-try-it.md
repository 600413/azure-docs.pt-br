
## <a name="launch-azure-cloud-shell"></a>Iniciar o Azure Cloud Shell

O Azure Cloud Shell é um shell interativo grátis que pode ser usado para executar as etapas neste artigo. Ele tem ferramentas do Azure instaladas e configuradas para usar com sua conta. Basta clicar em **Cópia** para copiar o código, colá-lo no Cloud Shell e depois pressione Enter para executá-lo.  Há duas maneiras de inicializar o Cloud Shell:

|  |   |
|-----------------------------------------------|---|
| Clique em **Experimente** no canto superior direito de um bloco de código. | ![Cloud Shell neste artigo](./media/cloud-shell-try-it/cli-try-it.png) |
| Clique no botão **Cloud Shell** no menu no canto superior direito do [portal do Azure](https://portal.azure.com). |    ![Cloud Shell no portal](./media/cloud-shell-try-it/cloud-shell-menu.png) |
|  |  |
