1. No menu do Visual Studio, selecione **Arquivo > Novo > Projeto**. 

1. Na seção **Modelos** da caixa de diálogo **Novo Projeto**, selecione **Visual C# > .NET Core**.

1. Selecione **ASP.NET Core Web Application (.NET Core) (Aplicativo Web do ASP.NET Core (.NET Core))**. ![Caixa de diálogo Novo Projeto](./media/vs-docker-create-aspnetcore-app/create-new-project.png)

1. Dê um nome ao novo aplicativo (ou use o padrão) e toque em **OK**.

1. Em **ASP.NET Core Templates (Modelos do ASP.NET Core)**, selecione **.NET Core**. ![Caixa de diálogo Novo Projeto ASP.NET](./media/vs-docker-create-aspnetcore-app/aspnet-core-template.png)

1. Desmarque **Host in the cloud (Host na nuvem)** já que você usará o Docker como sua solução de implantação.
